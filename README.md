# Automated-Code-Review-Debugging-Environment
This is a Multi-LLM RL Agent built for the Meta x OpenEnv Hackathon. It uses a Planner-Critic loop: ChatGPT (Actor) proposes actions, while Claude (Critic) refines them to prevent loops and errors. Built with Python and Pydantic, it features a custom Code-Debug environment, automated grading, and 2vCPU/8GB Docker optimization.
